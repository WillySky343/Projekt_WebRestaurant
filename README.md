# Projekt_WebRestaurant – La Bella Cucina

A simple, responsive restaurant website built with plain HTML, CSS and JavaScript — no frameworks or build tools required.

## Features

- **Sticky navigation** with smooth scrolling and a mobile hamburger menu
- **Hero section** with a call-to-action to explore the menu or book a table
- **Interactive menu** with tab-based category filtering (Starters, Pasta & Risotto, Main Courses, Desserts, Drinks)
- **About section** with restaurant story and key statistics
- **Photo gallery** with hover effects
- **Reservation form** with client-side validation (name, email, guest count, date, time)
- **Newsletter sign-up** form
- **Contact section** with opening hours and contact details
- **Responsive design** — works on desktop, tablet and mobile
- **Accessibility** — semantic HTML5, ARIA labels and keyboard navigation
- **Back-to-top button** and scroll-triggered fade-in animations

## Structure

```
Projekt_WebRestaurant/
├── index.html        # Main page
├── css/
│   └── style.css     # All styles (variables, layout, responsive)
├── js/
│   └── main.js       # Navigation, menu tabs, form validation, animations
└── README.md
```

## Running locally

Open `index.html` directly in any modern browser — no server or build step needed.
