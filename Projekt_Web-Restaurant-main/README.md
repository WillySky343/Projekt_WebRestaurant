# 🍝 Restaurant Reservierungssystem

Dieses Projekt entstand im Rahmen eines Universitätsprojekts und stellt eine vollständige Webanwendung für ein Restaurant dar.  
Die Anwendung ermöglicht es Nutzern, Reservierungen vorzunehmen, Benutzerkonten anzulegen und sich im System anzumelden.

---

## 🚀 Funktionen

- Benutzerregistrierung & Login  
- Tischreservierungen  
- Benutzerkontenverwaltung  
- Dynamische Inhalte mit Datenbankanbindung  
- Responsives Design

---

## 🛠️ Verwendete Technologien

Dieses Projekt wurde umgesetzt mit:

- **PHP** – Backend / Logik  
- **MySQL** – Datenbank  
- **HTML** – Struktur  
- **CSS** – Styling  
- **Bootstrap** – Layout & Responsivität  

---

## 📦 Installation

### Voraussetzungen

Für die Ausführung des Projekts werden benötigt:

- PHP ≥ 8.0  
- MySQL / MariaDB  
- Webserver (z.B. Apache, MAMP, XAMPP)
